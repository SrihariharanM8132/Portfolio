## ΏθγΑ Running Locally

### Option 1 έΑΦ Open directly (simplest)
Just double-click `index.html` έΑΦ no setup needed.

### Option 2 έΑΦ Live dev server (recommended, auto-reloads on save)

```bash

npm install

# Start the dev server
npm start
```

Then open **http://localhost:3000** in your browser.
